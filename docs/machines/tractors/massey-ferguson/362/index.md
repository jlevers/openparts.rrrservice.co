# Massey Ferguson 362

**Years:** 1988–1997  
**Engine:** Perkins A4.236 (4-cylinder diesel, 62 HP)  
**Transmission:** 8-speed (12-speed with optional creeper)  
**Drive:** 2WD and 4WD variants  

The MF 362 is a mid-range utility tractor from Massey Ferguson's 300-series. Shares many components with the 360, 365, 375, and 390 — so cross-reference notes are included throughout.

## Serial Number Notes

!!! warning "Know your serial number"
    Several components on the 362 changed at serial number **B33001**. The brake disc count, in particular, is different before and after this break point. Always confirm your serial number before ordering parts.

## Common Cross-Applications

The MF 300-series shares a lot of parts. These models frequently interchange components with the 362:

- **MF 340, 342, 350, 352, 355** — Many brake and hydraulic components
- **MF 360, 365** — Near-identical machines
- **MF 372, 375, 382, 383** — Shared brake master cylinders and some hydraulic parts
- **MF 390, 390T, 398, 399** — Brake master cylinder, some drivetrain components

Cross-application is noted on each parts page where confirmed.
