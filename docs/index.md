# OpenParts

**Open-source mechanical parts database. Built by mechanics, for mechanics.**

## What's Here

OpenParts is organized by machine type, manufacturer, model, and system:

- **[Machines](machines/index.md)** — Browse the full directory
- **[Tractors](machines/tractors/index.md)** — Tractor parts database
- **[Massey Ferguson 362](machines/tractors/massey-ferguson/362/index.md)** — Complete parts data for the MF 362

We're just getting started. The database grows every time someone contributes what they know.

## How to Contribute

OpenParts lives on GitHub. If you've got parts knowledge to share:

1. **Fork the repo** → [jlevers/openparts.rrrservice.co](https://github.com/jlevers/openparts.rrrservice.co)
2. **Add or edit** parts data in Markdown
3. **Submit a pull request** — we'll review and merge

No special tools needed. If you can write a text file, you can contribute. See the **[Contributing Guide](contributing.md)** for details on format and what makes a good entry.

!!! tip "Don't have a GitHub account?"
    Open an issue on the repo with your data and someone will format it. The knowledge matters more than the workflow.
